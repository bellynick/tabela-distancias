### Descrição
O projeto consiste na composição de uma tabela que conta com capitais Brasileiras e a distância entre essas capitai. As capitais contastantes nesse projetos são:

* Aracaju
* Belém
* Horizonte
* Boa Vista
* Brasilia
* Campo grande
* Cuiabá
